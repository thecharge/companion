# Source Header Policy

Use this source header for new TypeScript files:

```ts
/*
 * SPDX-License-Identifier: MIT
 * Copyright (c) 2026 Companion contributors
 */
```

Notes:
1. Keep SPDX identifier aligned with `LICENSE`.
2. Apply to new source files and touched critical files incrementally.
3. Enforce with lint check in CI when ready.
